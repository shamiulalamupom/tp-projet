# Demarrer l'application

Guide rapide pour installer les dependances et lancer l'application en local.

## Prerequis

- Node.js 18.19+ (LTS 20 recommande)
- npm (installe avec Node.js)
- Angular CLI 21.0.3

## Installation

Dans le repertoire du projet, installer les dependances :

```bash
npm install
```

## Lancement en developpement

- Demarrer le serveur local :

```bash
ng serve
```

- Ouvrir `http://localhost:4200/` dans le navigateur; le rechargement a chaud est actif.
- Arreter le serveur avec `Ctrl+C` dans le terminal quand vous avez fini.
